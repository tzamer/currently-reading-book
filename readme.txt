A simple wordpress plugin that shows the current book you are reading along with book preview and book information.

== Description ==

This is a simple wordpress plugin that shows which book(s) you are currently reading. It uses the Google Books API, and can show the covers as well as the previews of the books.

What this plugin offers:

*   Look a book up by its ISBN number
*   Show the book cover, including the book information you choose (book title, author, ISBN number)
*   Add your thoughts on the current book (optional)
*   Show the previously read book (optional)
*   Supports widget mode.

== Installation ==

1. Upload the plugin directory to to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `<?php currently_reading_book(); ?>` in your templates or add the widget 'Currently Reading Book' to the theme you are currently using
